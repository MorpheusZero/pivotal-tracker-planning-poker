Pivotal Tracker - Planning Poker Exporter

This tool was created so that a user can export Pivotal Tracker stories into the CSV Format that Planning Poker uses for imports.

Step 1: Pivotal Tracker API Key

Go to your profile page in Pivotal Tracker to obtain your API Key. Copy/Paste it into the box below.

Step 2: Project ID

Copy/Paste Your Project ID into the box below.

Step 3: Tag Name

Copy/Paste the name of the tag you want to include in the stories returned. This can be useful for if you tag the stories ahead of time like "to-be-scored" or something.

Step 4: Get Stories

Press the button below to get the stories that match your search criteria.

Review Data

These are the stories that were returned according to your search criteria. If this looks correct, press EXPORT.

The data below is in the correct CSV format that Planning Poker requires. You can copy the data below and paste it into a text editor and "Save As..." .csv and then import that file into Planning Poker!
